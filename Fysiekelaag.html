Fysiekelaag

Wat is RAM?

RAM staat voor Random Access Memory. Het is een type geheugen in je computer dat tijdelijk gegevens opslaat die je computer nodig heeft om snel te kunnen werken. RAM is veel sneller dan andere vormen van opslag, zoals harde schijven of SSD's.

Hoe werkt RAM?

Wanneer je een programma opent of een bestand bewerkt, worden de gegevens van dat programma of bestand tijdelijk opgeslagen in RAM. Dit gebeurt omdat RAM veel sneller toegang biedt dan andere opslagmedia. Terwijl je werkt, kan je computer snel gegevens ophalen en verwerken vanuit RAM, wat zorgt voor een soepelere en snellere werking van je computer.

RAM is vluchtig, wat betekent dat het geheugen leeg raakt zodra je je computer uitschakelt. De gegevens die je in RAM hebt opgeslagen, gaan verloren als je de computer afzet of opnieuw opstart.
